\# Online Retail Sales Database Design



\## Objective

To design and implement a normalized relational database for an online retail system using SQL.



\## Tools Used

\- MySQL

\- dbdiagram.io

\- GitHub



\## Project Features

\- Normalized schema (3NF)

\- Primary \& Foreign key constraints

\- Sample data population

\- JOIN queries for reporting



\## Database Entities

\- Customers

\- Products

\- Orders

\- Order\_Items

\- Payments



\## How to Run

1\. Open MySQL Workbench

2\. Run `retail\_db.sql`

3\. Execute queries from `Query\_Report.sql`
